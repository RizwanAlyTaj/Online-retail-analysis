# Online Retail Data Analysis

## Overview
End-to-end data analysis on a 10,000-row e-commerce
dataset as part of the Essential Software course.

## Steps Covered
- Data loading and exploration
- Data cleaning — missing values, duplicates, outliers
- Univariate EDA — histograms, boxplots, bar and pie charts
- Bivariate analysis — scatterplots and correlation heatmap
- Market overview — revenue trends and business insights
- Business recommendations after every finding

## Key Findings
- Quantity is the strongest revenue driver (correlation 0.56)
- UK accounts for ~95% of all transactions
- November-December is the peak revenue season
- Volume strategy beats price increase strategy

## Tools Used
Python · pandas · numpy · matplotlib · seaborn · Jupyter
